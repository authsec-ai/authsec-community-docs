# Logs Configuration

Configure output destinations for your logs: Splunk, HTTP endpoints, Syslog servers, or Elasticsearch.

Choose from the following log destinations:

- **[Splunk](./splunk.md)** - Connect via HTTP Event Collector
- **[Fluent Bit](./fluent-bit.md)** - Stream logs for processing and forwarding
- **[Syslog](./syslog.md)** - Stream to Syslog server via UDP or TCP
- **[Elasticsearch](./elasticsearch.md)** - Index logs directly into Elasticsearch
