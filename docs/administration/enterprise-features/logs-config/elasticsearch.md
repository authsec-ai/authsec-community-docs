# Elasticsearch

Index logs directly into Elasticsearch.

The Elasticsearch output plugin allows you to ingest your records into an Elasticsearch database. It supports template management, HTTP authentication, and TLS/SSL encryption.

## Configuration Steps

1. Tenant ID is automatically filled from your session
2. Enter domain (e.g., `logs.example.com`) or IP address (e.g., `192.168.1.100`)
3. Port field appears only for IP addresses
