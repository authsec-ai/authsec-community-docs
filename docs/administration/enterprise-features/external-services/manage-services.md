# Manage External Services

## Service List

Each external service entry displays:

- **Service name**
- **Service type**
- **Credential type**
- **Available SDKs**
- **Actions menu**

## Available Actions

![Edit Service](/screenshots/external-service/edit-service.png)

### View SDK Details

Access integration details and SDK documentation for the service.

### Edit Service Configuration

Modify service settings including:
- Service name
- API URL
- Authentication credentials
- Tags and description

### Delete Service

Permanently remove a service integration.

**Note**: Deleting a service will remove all associated credentials and configurations. This action cannot be undone.
