# External Services and Secrets Management

## Overview

The External Services page allows you to manage integrations with third-party APIs and external systems. It provides a secure way to connect services using OAuth or API keys, store secrets safely in a vault, and control access at runtime.

![External Services Overview](/screenshots/external-service/overview.png)

This page is used to manage OAuth connections, API credentials, and service-to-service authentication.

## Key Capabilities

- **Secure OAuth token storage in a vault**
- **Service-to-service authentication**
- **Automated token refresh and lifecycle management**
