# Sync from Directory

1. Open the Users page in the dashboard
2. Click the **Add User** button
3. Select **Active Directory** or **Entra ID**
4. Configure the directory connection:
   - Server details
   - Credentials
   - Base DN or Tenant information
5. Click **Test Connection**
6. Click **Sync Users**

Users are imported automatically.
