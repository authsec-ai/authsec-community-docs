# Users

## Overview

Users are the individuals who access your application through AuthSec. User management allows you to control who can authenticate, what roles they have, and how they can access your system. You can add users manually by invitation or sync them automatically from your organization's directory (Active Directory or Entra ID).

## Prerequisites

1. Access to the AuthSec dashboard
2. Admin permissions to manage users
3. A configured tenant
