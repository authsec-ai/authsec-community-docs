# Step 3: Create Scopes

## What are scopes?

Scopes define the level of access granted by a permission. Common scopes include 'read' (view only), 'write' (create/edit), 'delete' (remove), and 'admin' (full control). They work with roles to create granular permissions.

## Steps

1. Open the **Scopes** page in the dashboard
2. Click **"Create Scope"**
3. Define the scope details:
   - Name: `read`, `write`, `delete`, or `admin`
   - Description: Brief explanation of the access level
4. Configure scope permissions
5. Save the scope
