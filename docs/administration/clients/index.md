# Clients Page – Set Up Clients (MCP Servers / AI Agents)

## Overview

The Clients page allows you to set up and manage multiple clients to represent MCP servers or AI agents. Each client helps separate different functionalities across agents and servers within the same project.

This setup enables fine-grained authentication control, isolation, and scalability for AI-driven systems.
