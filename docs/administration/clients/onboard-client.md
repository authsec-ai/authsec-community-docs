# Onboard Client

You can onboard a new client to represent an MCP server or an AI agent.

## Steps to Onboard a Client

![Onboard Client](/screenshots/clients/onboard-client.png)

1. Click the **"Onboard Client"** button
2. Enter a unique client name (for example, `My MCP Server`)
3. Click **Onboard**

A new client is created and appears in the clients list with a unique **Client ID**.

## Client List and Details

Each client entry displays:

- **Client name**
- **Client ID**
- **Status** (Active or Inactive)
- **Enabled authentication methods**
- **Total number of users**
- **Action menu**

Filters and search allow you to find clients by name or endpoint and filter by access level.
