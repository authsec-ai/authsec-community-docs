# Authentication Setup

## Overview

The Authentication page allows you to set up authentication methods for end users using Single Sign-On (SSO).

Available authentication options:

- **OIDC / OAuth 2.1** (Google, GitHub, Microsoft)
- **SAML 2.0** (Okta, Entra ID)

You can manage all authentication providers from the Authentication Page in the UI.

![Authentication Page](/screenshots/auth-methods/1.png)

## Related Documentation

For machine-to-machine or agent authentication, refer to the [autonomous agent authentication](/autonomous-agents) documentation.

## Set up Authentication Methods

- **[OIDC / OAuth 2.1](./oidc-setup)** - Configure Google, GitHub, or Microsoft authentication
- **[SAML 2.0](./saml-setup)** - Configure Okta or Entra ID authentication
- **[Manage Providers](./manage-providers)** - Deactivate or delete authentication methods
