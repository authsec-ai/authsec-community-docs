# AuthSec SDK Documentation

Welcome to the AuthSec SDK documentation. Our SDKs provide programmatic access to AuthSec services, enabling you to integrate authentication, authorization, and workload identity management into your applications.

---

## Available SDKs

### Clients SDK

Integrate AuthSec authentication into your client applications:

- **OpenID Connect** integration
- **SAML** authentication support
- **Session management** utilities

[**Explore Clients SDK →**](./clients/mcp-servers.md)

---

### Workloads SDK

Secure your autonomous workloads and machine-to-machine communication:

- **Workload identity** management
- **Certificate rotation** automation
- **Zero-trust** attestation

[**Explore Workloads SDK →**](./workloads/autonomous-workloads.md)

---

### RBAC SDK

Implement role-based access control in your applications:

- **Role management** APIs
- **Permission** enforcement
- **Policy** evaluation

[**Explore RBAC SDK →**](./rbac/permissions-resources.md)

---

### External Services SDK

Integrate with external authentication providers and services:

- **Third-party** provider integration
- **Identity federation** support
- **API** connectors

[**Explore External Services SDK →**](./external-services/secret-management.md)

---

## Getting Started

1. **Choose your SDK** based on your use case
2. **Install** the SDK using your package manager
3. **Configure** authentication credentials
4. **Follow** the integration guides for your platform
