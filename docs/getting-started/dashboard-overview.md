# Dashboard Overview

Once logged in, you will land on the **AuthSec Dashboard** - your central hub for managing authentication, users, and security.

![AuthSec Dashboard](/screenshots/getting-started/8.png)

---

## Dashboard Features

### Overview Metrics

Get real-time insights into your authentication environment:

| Metric | Description |
|--------|-------------|
| **Active Sessions** | Currently logged-in users |
| **Total Users** | All registered users in your workspace |
| **Admin Users** | Users with administrative privileges |
| **Inactive Sessions** | Expired or logged-out sessions |

These metrics help you monitor user activity and identify potential security concerns.

---

### Integration Tours

Quick-start guides to help you configure key features:

| Tour | What You'll Configure |
|------|----------------------|
| **User Authentication Setup** | Configure OIDC / SAML for your applications |
| **M2M Auth** | Machine-to-machine authentication for APIs and services |
| **RBAC Setup** | Roles and permissions for fine-grained access control |
| **Scopes Setup** | Permission boundaries for API access |

These interactive tours guide you through each setup process step-by-step.

---

### Quick Actions

Commonly used features accessible with one click:

- **Sync Active Directory users** - Import users from your organization's directory
- **Configure authentication providers** - Add OAuth, SAML, or OIDC providers
- **Manage secrets and integrations** - Store API keys and credentials securely
- **View authentication logs** - Monitor sign-in attempts and security events

---

## Navigation

The dashboard provides easy access to all major sections:

### Left Sidebar Navigation:

- **Home** - Dashboard overview
- **Users** - User management and provisioning
- **Administration** - Clients, RBAC, and workspace settings
- **Autonomous Agents** - M2M authentication configuration
- **Enterprise Features** - Logs, compliance, and advanced settings
- **Settings** - Workspace configuration and preferences
