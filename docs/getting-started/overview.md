# Getting Started

This section covers first-time setup for an AuthSec account and workspace.

## AuthSec overview

AuthSec includes the following capabilities:

- Email and OAuth sign-in
- Multi-factor authentication (MFA), including WebAuthn
- Role-based access control (RBAC)
- Machine-to-machine authentication
- Audit logs and external secret integrations

## What this guide covers

In this guide, you will:

1. Create or sign in to your account
2. Verify your email address
3. Create your workspace
4. Configure MFA
5. Sign in and access the dashboard
6. Review next steps for integration

## Before you begin

Make sure you have:

- Access to a work email or OAuth provider account
- An authenticator app or security key (if MFA is required)
- Admin access for your organization (for workspace setup)

Continue with [Sign In Setup](/getting-started/sign-in-setup).
