# Authsec Community Docs

Community-maintained documentation for [Authsec](https://github.com/authsec) — open-source authentication and authorization for AI Agents & MCP servers.

Built with [Docusaurus](https://docusaurus.io/).

## Getting Started

### Prerequisites
- Node.js >= 20.0

### Run locally

```bash
npm install
npm run start
```

This starts a local dev server at `http://localhost:3000` with hot reload.

### Build

```bash
npm run build
```

Generates a static site in the `build/` directory.
